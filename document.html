URL Utilizada
URL: https://pokeapi.co/api/v2/pokemon/

Link da API utilizado para o consumo de dados dos POKÉMONs.

Não foi preciso fazer nenhum cadastro para conseguir a chave para o consumo.


Para mais informações sobre a URL entre no site: https://pokeapi.co/

Possíveis Problemas
  • Não conseguir resgatar os dados dos Pokémons.
  • Erro com algumas imagens.
  • As imagens dos Pokémons está como atributo SPRITES.
Soluções
  • Na URL passar o id do Pokémon que irá diretamente nos dados do Pókemon específico.
  • Algumas imagens Pokémons não existe na API.
  • Procurar por sprites quando estiver nos dados de um Pokémon específico.
